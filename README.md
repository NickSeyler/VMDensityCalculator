# VMDensityCalculator

A VM Density Calculator coded in python.
<br />

Inputs:
<br />

Processor Quantity

Number of Cores

Gigahertz

RAM Per Host
<br />

vCPU to Core Ratio

Cores assigned to Hypervisor

Ram Assigned to Hypervisor

Hyperthreading (yes or no)
<br />

Guest vCPUs Assigned

RAM per Guest
<br />

Outputs:
<br />

Total Number of VMs

Constraint

Cores per VM

Gigahertz per VM
