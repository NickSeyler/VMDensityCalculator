# VMDensityCalculator

A VM Density Calculator coded in python.
&nbsp;

Inputs:
&nbsp;

Processor Quantity

Number of Cores

Gigahertz

RAM Per Host
&nbsp;

vCPU to Core Ratio

Cores assigned to Hypervisor

Ram Assigned to Hypervisor

Hyperthreading (yes or no)
&nbsp;

Guest vCPUs Assigned

RAM per Guest
&nbsp;

Outputs:
&nbsp;

Total Number of VMs

Constraint

Cores per VM

Gigahertz per VM
