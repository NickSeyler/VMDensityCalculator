# VMDensityCalculator

A VM Density Calculator coded in python.


Inputs:

Processor Quantity

Number of Cores

Gigahertz

RAM Per Host


vCPU to Core Ratio

Cores assigned to Hypervisor

Ram Assigned to Hypervisor

Hyperthreading (yes or no)


Guest vCPUs Assigned

RAM per Guest

Outputs:

Total Number of VMs

Constraint

Cores per VM

Gigahertz per VM
